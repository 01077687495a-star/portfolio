#  **[TC-02] 공격 속도 증가에 따른 스킬 액션 프레임 단축 확인**
---
Status: 🟡 In-Progress 

## **개요**

공격 속도 증가 시 특정 임계 값에서만 스킬 액션 프레임이 단축되는 것이 아닌 선형적으로 단축되는 것을 확인.

특정 임계 값을 기준으로 공격속도가 크게 차이가 나는지, 같은 공격속도를 갖던 구간의 양 끝 값을 비교해 공격횟수가 같은지 차이가 있는지 확인.

1분동안 제자리에서 스킬을 사용하며 사용한 스킬 횟수 카운트

**참고 자료**

**공격 속도의 효과가 정확한 수치대로 반영되기 전의 공격속도 적용 로직(버그 패치 전)**
| 단계 (Stage) | 공격 속도 구간 | 판정 기준 |
| :--- | :--- | :--- |
| **Stage 0** | 0% ~ 11.09% | 기본 공격 속도 유지 |
| **Stage 1** | 11.1% ~ 24.99% | 1단계 속도 상승 (모션 프레임 단축) |
| **Stage 2** | 25% ~ 42.85% | 2단계 속도 상승 |
| **Stage 3** | 42.86% ~ 66.66% | 3단계 속도 상승 |
| **Stage 4** | 66.67% ~ 99.99% | 4단계 속도 상승 |
| **Stage 5** | 100% 이상 | 5단계 속도 상승 (최종 단계) |



## **사전 조건**

변수차단을 위해 자동클릭 매크로 사용 (Macrodroid 사용)

공격 속도 스탯을 자유롭게 조절 가능한 테스트 환경

## **제약 조건**

실제 테스트 계정에서는 스탯을 자유롭게 조절 가능하지 못해 모든 테스트가 실질적으로 불가했음.
