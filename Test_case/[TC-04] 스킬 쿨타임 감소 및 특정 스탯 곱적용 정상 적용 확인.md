#  **[TC-04] 스킬 쿨타임 감소 및 특정 스탯 곱적용 정상 적용 확인**
---
Status: ⚪ N/A

## **개요**

스킬 쿨타임 감소와 특정 스탯의 곱적용 연산이 제대로 적용되는지를 확인한다.

## **사전 조건**

여러 스탯을 자유롭게 조절할 수 있는 테스트 환경

## **제약 조건**

실제 테스트 계정에서는 스킬 쿨타임 관련 옵션을 보유하고 있지 않고 특정 스탯을 상한선에 가깝게 보유하기 힘들어 실질적인 테스트가 불가했음
